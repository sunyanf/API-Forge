# API-Forge - 为独立开发者打造的AI原生API工坊

## 1. 项目愿景
成为一个“小而美”的API服务平台，让开发者能一站式获取实用数据接口和先进的AI内容生成能力。我们通过清晰的服务分级，展示AI如何在商业场景中创造真实价值。

## 2. 目标用户画像
-   **个人开发者/独立创业者**：需要快速集成基础功能，并低成本尝试AI能力来武装自己的产品。
-   **面试官/技术审查者**：重点展示项目主人在后端架构、商业思维和AI应用三个维度的综合能力。

## 3. 核心功能（最小可行产品 MVP）

### 基础服务层（对所有注册用户免费）
-   **用户系统**：邮箱注册、登录、API Key生成与管理。
-   **IP归属地查询API**：输入IP，返回国家、城市、运营商信息。
-   **随机图片API**：每次请求返回一张精美图片URL，可用于前端占位。

### 增值服务层（仅对付费/高级用户开放）
-   **AI内容创作API**：
    -   **接口**：`POST /api/v1/ai/content`
    -   **功能**：接收用户的关键词和内容类型（如“小红书文案”、“产品卖点”），调用大模型，返回高质量、可商用的文案。
    -   **计费模式模拟**：通过API Key的服务等级，模拟扣费或限制调用次数。

### 平台管理功能
-   **API调用日志**：所有接口的调用历史，便于开发者自查。
-   **使用量看板**：一个简单的可视化图表，展示基础API和AI API的调用次数对比。

## 4. 核心价值与简历叙事
**一句话总结**：独立全栈开发了一个模拟商业化的AI原生API平台，实现了从用户鉴权、基础服务到AI增值服务的完整业务闭环。

**面试时可引导的技术亮点**：
-   **商业架构设计**：我设计了免费与付费的服务分级体系，这让我深入思考了SaaS产品的商业模式。
-   **AI工程化落地**：我不仅调用了AI模型，还围绕它构建了产品（Prompt管理、计费模拟、日志监控），将AI能力产品化。
-   **全栈工程能力**：独立完成Go后端分层架构、MySQL数据库设计、前端可视化看板及Docker容器化部署。
